# Movie Finder

[**Live Site**](https://movie-finder-sigma-two.vercel.app/)

### Description
A site for exploring your passion for cinema.

### Tech Stack
React, React Router, Vite, TailwindCSS

### Why
The purpose of this project was to practice using React Router and fetching data from APIs. I also enjoy the history of cinema and finding quirky films.

### Features
* User input to direct searches
    * Input validation
* User selected sorting of results
* Fetching data from remote APIs
* Pagination of results
